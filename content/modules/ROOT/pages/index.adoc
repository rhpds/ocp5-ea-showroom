Welcome to the *OpenShift 5 Early Access* hosted environment.

This guide will walk you through the new features available in OpenShift 5
as part of the Early Access program.

== Getting Started

Your environment has been provisioned with:

* An OpenShift 5 cluster
* Red Hat Advanced Cluster Security (ACS)
* OpenShift GitOps (Argo CD)
* OpenShift Virtualization with a RHEL 9 VM
* OpenShift Lightspeed (AI assistant)
* Cluster Observability Operator with Perses dashboards
